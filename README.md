## Я Евгений Кононенко, frontend разработчик

Уже более 3 лет в коммерческих проектах: веб (SPA, PWA, SSR), мобильные приложения на React Native и десктоп. Основной стек: TypeScript, React, Vue и их экосистемы.

- Реализовывал продуктовые фичи: корпоративный мессенджер, модуль пополнения баланса через СБП, PWA-миграция

- Занимался оптимизацией производительности: Core Web Vitals, рендеринг, скорость загрузки страниц
- Делал мобильные приложения на React Native (CLI/Expo) - от архитектуры проекта до готового продукта
- Запустил десктопное приложение на Electron: нативные C++ модули, интеграция с фискальным оборудованием АТОЛ/PAX
